About

The Tools Registry will catalogue Web services and other software required by the BioMedBridges partners and other bioinformatics and biomedical researchers. This is the prototype: it's purpose is to test ideas and get feedback for the first beta version.

Please help by sending ideas and feedback to Jon Ison cc'ing Julie McMurry.

Alpha version 06

New features for alpha version 6 include:

  • Clean grouping of entries via collections - any entry can be a collection. For example, a database might include (collect) various Web applications, a tool such as a Web application might provide (collect) distinct modes of functionality - each of which appears as a distinct registry entry.
