About

The Tools and Data Services Registry will catalogue Web services and other software required by the BioMedBridges partners and other bioinformatics and biomedical researchers. This is the prototype: it's purpose is to test ideas and get feedback for the first beta version.

Please help by sending ideas and feedback to Jon Ison cc'ing Kristoffer Rapacki and Julie McMurry.

Alpha version 08

This version is a major content release and includes 675 new databases (imported from DRCAT).

Each database is annotated with:

  • Name
  • Homepage
  • Short description
  • EDAM Topic(s)

Some minor cosmetic changes:

  • Repositioned search box and Show Menu
  • New columns visible by default
  • Cleaner search button

See our plans for the next release.
