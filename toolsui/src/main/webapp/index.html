Alpha version 03


The BioMedBridges registry will catalogue Web services and other software required by the BioMedBridges partners. It is developed under ELIXIR auspices. This is the prototype. It's purpose is to test ideas and get feedback for the first beta version. Please help by sending ideas and feedback to Jon Ison cc'ing Julie McMurry. See our plans for the next release.
