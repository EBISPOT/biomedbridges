Contribute

This page will describe how you can contribute to the Tools and Data Services Registry. For now, if you'd like to contribute please email our helpdesk.

Descriptions of tools of relevance to BioMedBridges are currently held in a Google spreadsheet at tinyurl.com/bmbtools: if you are a BioMB partner, please feel free to edit this spreadsheet.

In future the expectation is that tool providers will maintain their own tool descriptions which will be imported into the registry. For this purpose we will support various mechanisms, including:

  • Bulk registration of multiple tools via XML. Please see our Software Description Model
  • Registration of one or a few tools via a Registration Interface (in development)
  • Any other means, at your convenience
