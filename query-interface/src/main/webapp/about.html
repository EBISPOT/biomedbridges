About

The Tools and Data Services Registry will catalogue Web services and other software required by bioinformatics and biomedical researchers, especially those working within one of the ten biomedical sciences research infrastructures of BioMedBridges.

A complete dump of the XML is available here: http://tinyurl.com/bmbRegistryXMLContent.

It validates to a prototype schema, available here: http://wwwdev.ebi.ac.uk/fgpt/toolsui/schema.

Query Interface Beta version 03

This includes cosmetic and other changes to improve usability and new features:

  • New "Export" menu supports download of filtered data in a variety of formats (2MB limit, due to JQuery licensing)
  • Major revision to "Show" menu entries making this more intuitive and usable
  • Search box and "Show more..." (columns menu) button now more visible (resized and repositioned)
  • Grid columns are more compact, with new ions rather than text for links, where possible
  • Sort menus removed from columns (now just click the menu header)
  • All tooltips improved (clearer meaning) and better positioned
  • "Partners" now includes BioMedBridges infrastructures and other partners
  • Various minor bug fixes (disappearing "Partners" menu, unselectable "Type" from drop-down, text alignment on moving columns

Please see feedback from this query interface for a summary of features that will be prioritized in a new version of the registry now in development.

Please help by providing feedback to the BMB registry helpdesk..
