About

December 2014: This is an archived version of the registry prototype developed for BioMedBridges in partnership with ELIXIR.

This prototype registry catalogues Web services and other software required by bioinformatics and biomedical researchers, especially those working within one of the ten biomedical sciences research infrastructures of BioMedBridges.

  • Query Interface: Informed by this BMB prototype (Beta version 03), ELIXIR is developing a new generation Tools and Data Services registry (coming soon).
  • XML content: A complete archive of the BMB prototype XML is available here.
  • Data model: This is prototype XML, which validates to the BMB prototype schema ; all future releases of this data will conform to the new ELIXIR schema and will be searchable/programmatically accessible within the ELIXIR version.

    • Feedback and contributions

  • Any new contributions of XML content should conform to this new schema; see new ELIXIR schema for details.
  • Please see feedback from this query interface for a summary of features that will be prioritized in a new version of the registry now in development within ELIXIR.
  • Please help by providing feedback to the BMB registry helpdesk.
