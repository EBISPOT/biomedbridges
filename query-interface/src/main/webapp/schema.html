Software Description Model

Please note that the XML available in this current version of the interface (beta_03) is compliant to a prototype schema available here.

Both the schema and the interface have since undergone extensive re-development; consequently the the prototype schema is not meant to be prescriptive for new contributions of content.

For contributions of new content, please instead refer to the latest version of the schema, now available at http://bioregistry.cbs.dtu.dk/schema.html. A corresponding new version of the query interface will be available soon.

Details about our data policy are available here.

For all enquiries on using this schema please contact our helpdesk.
